IoT Club


YOU DO NOT TALK ABOUT THE IOT CLUB.

Rules

Thou shalt not design an insecure web interface.

Thou shalt provide sufficient authentication and authorization.

Thou shalt provide secure network services.

Thou shalt encrypt thy traffic.

Thou shalt provide sufficient security configurability.

Thou shalt verify the security of thy firmware and patch software if need be.

Thou shalt provide a secure mobile interface.

Thou shalt host data on a secure Cloud.

Thou shalt not use or sell personal or private information for unethical purposes.

Thou shalt ensure thy shipped device is physically secure.
